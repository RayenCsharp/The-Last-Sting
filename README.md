# The Last Sting

A 3D third-person action game developed in **Unity 6** for the **Made To Be Played**.

> **Game Jam:** Made To Be Played  
> **Jam Page:** [<Link>](https://itch.io/jam/made-to-be-played)

---

## About

The Last Sting is a third-person action game where you play as a scorpion trapped inside a closed garden. Fight hostile insects using fast melee attacks and powerful venom abilities while managing your poison resource to survive.

This project was developed as a solo game jam project and serves as a learning experience in gameplay programming, AI, animation, and low-poly 3D art.

---

## Theme

**Toxic**

The theme is represented through the scorpion's venom-based combat, poisonous abilities, and the dangerous ecosystem of the garden.

---

## Features

- 🦂 Third-person scorpion controller
- ⚔️ Two-hit melee combo
- ☣️ Venom sting special attack
- 💚 Health pickup system
- 🟣 Poison resource system
- 🤖 Enemy AI with multiple enemy variants
- 🐜 Multiple insect enemies
- 🌳 Low-poly garden environment
- 🎨 Models and animations created in Blender

| Look | Mouse |
| Jump | Space |
| Sprint | Left Shift |
| Normal Attack | Left Mouse Button |
| Venom Attack | Left Ctrl + Left Mouse Button |

## Development Notes

This was my **first game jam project** and my first complete 3D game developed within a limited time frame.

Although I missed the official submission deadline, I continued working on the project to complete it as a personal challenge and learning experience. Throughout development I learned about player controllers, enemy AI, animation systems, combat mechanics, Unity's Input System, Blender low-poly modeling, and project organization.

This project represents an important milestone in my journey as a game developer.
